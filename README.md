# 人生博物馆 · Museum of Lives

把一本人物传记，变成一座可漫游的博物馆。

## 项目简介

人生博物馆是一个多媒体传记平台，将传统人物传记从"读一本书"升级为"逛一座馆"——读者可以通过时间轴、地图、人物关系网络、金句长廊等交互方式，沉浸式探索一个人的一生。

### 首批馆藏

- **埃隆·马斯克传**：深空美学主题，展示从南非少年到火星使者的冒险人生
- **苏东坡传**：宋韵雅致主题，呈现从眉山少年到岭海流放的豁达人生

## 技术栈

- React 18 + Vite + TypeScript
- TailwindCSS 3
- React Router DOM
- Leaflet + react-leaflet（真实地图）
- Lucide React（图标）
- Canvas API（粒子背景）

## 功能模块

### 平台层（博物馆首页）
- Hero 品牌区
- 馆藏卡片入口
- 群星闪耀精选（跨传记决定性时刻）
- 理念阐述区
- 命运岔路口精选（反事实推演）
- 人物星图预览
- 数据仪表盘
- Footer

### 传记层（统一9大模块）
1. Hero 全屏开场
2. 导读（三段叙事 + 金句）
3. 人生时间轴
4. 成就版图（商业帝国 / 传世佳作）
5. 真实地图（人生足迹）
6. 人物星图（关系网络）
7. 金句廊
8. 戏剧时刻（命运临界点）
9. 命运岔路口（反事实模块）
10. 主题深度
11. 尾声

## 设计理念

- **沉浸式阅读**：通过星空/水墨粒子背景、滚动触发动画、玻璃态卡片打造沉浸体验
- **多维度叙事**：时间轴（纵向）、地图（横向）、人物关系（网状）多角度呈现人物
- **反事实思考**："命运岔路口"模块引导读者思考历史的偶然性
- **平台化扩展**：统一数据 Schema，新增传记只需替换数据文件
