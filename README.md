# 人生博物馆 · Museum of Lives

> 不止于读 · 让每一段人生都可触摸、可分叉、可对照

把一本人物传记，变成一座可漫游的博物馆。

## 项目简介

人生博物馆是一个多媒体传记平台，将传统人物传记从"读一本书"升级为"逛一座馆"——读者可以通过时间轴、真实地图、人物关系网络、金句长廊、命运岔路口等交互方式，沉浸式探索一个人的一生。

## 核心亮点

### 🎨 沉浸式多维度叙事
- **时间轴**：纵向展开人生的每一个关键节点
- **真实地图**：Leaflet 瓦片地图，钉住每一段足迹
- **人物星图**：三层同心圆关系网络，看清谁影响了谁
- **戏剧时刻**：命运临界点，感受历史的张力
- **命运岔路口**：反事实推演，如果当时不是这样……

### ⚙️ 传记工坊 · 可复用 Skill
内置 `biography-parser` Skill，实现从 PDF 到博物馆的标准化转化：

```
PDF 投入 → 10 维数据提取 → 组件化生成 → 主题适配 → 路由注册 → 版权声明
```

- **10 维数据模型**：章节 / 时间轴 / 金句 / 人物星图 / 人生足迹 / 戏剧时刻 / 命运岔路口 / 核心主题 / 成就版图 / 地图路径
- **6 步标准流程**：一本新书几小时内即可入库
- **可无限扩展**：每一本新传记都沿用同一套骨架，持续生长

### 🌌 深夜博物馆视觉
深墨蓝底色 + 鎏金点缀 + 玻璃态卡片，每本传记有独立主题色和粒子背景风格。

## 技术栈

- **框架**：React 18 + Vite + TypeScript
- **样式**：TailwindCSS 3
- **路由**：React Router DOM
- **地图**：Leaflet + react-leaflet（CartoDB Dark Matter 瓦片）
- **图标**：Lucide React
- **动画**：Canvas API（粒子背景）+ CSS 动画（滚动入场）
- **部署**：GitHub Pages + GitHub Actions 自动构建

## 功能模块

### 平台层（博物馆首页）
- **Hero**：品牌开篇
- **Philosophy**：核心理念 — 把传记变成可漫游的博物馆
- **BiographyWorkshop**：传记工坊 — 从 PDF 到展厅的转化过程
- **Dashboard**：馆藏一览
- **Biographies**：馆藏传记卡片入口
- **ConstellationPreview**：人物星图预览（4 段人生交相辉映）
- **Footer**：收尾

### 传记层（统一 11 模块骨架）
1. Hero 全屏开场
2. 序章·导读（作者其人 → 传主其人 → 为何值得读）
3. 人生时间轴（3-5 阶段，30-50 事件）
4. 成就版图（商业帝国 / 传世佳作 / 功勋事业）
5. 人生足迹（真实地图 · Leaflet）
6. 人物星图（三层同心圆关系网络）
7. 金句廊（15-20 条经典语录）
8. 戏剧时刻（命运临界点 · 含反事实推演）
9. 命运岔路口（5 个关键决策 · 平行历史切换）
10. 主题深度（4-6 个核心主题）
11. 尾声 + 版权声明

## 设计理念

- **沉浸式阅读**：星空/水墨/银点粒子背景、滚动触发动画、玻璃态卡片
- **多维度叙事**：时间轴（纵向）、地图（横向）、人物关系（网状）多角度呈现
- **反事实思考**："命运岔路口"模块引导读者思考历史的偶然性
- **平台化扩展**：统一数据 Schema + 可复用 Skill，新增传记只需投入一本 PDF
